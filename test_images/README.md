# Test Images

This folder contains test images for evaluating the food classification model.

## How to Add Test Images

1. Download sample food images from Food-101 dataset or use your own food photos
2. Place them in this folder
3. Use them to test the classification accuracy in the application

## Recommended Test Images

You can download sample images from:
- Food-101 dataset: https://www.kaggle.com/datasets/kmader/food41
- Or use images from categories like:
  - apple_pie
  - pizza
  - hamburger
  - sushi
  - steak
  - etc.

## Usage

Upload these images through the web interface at `/` to test the classification model.
